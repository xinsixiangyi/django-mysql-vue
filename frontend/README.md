# frontend

> A Vue.js+django+mysql project

# require
>django==2.0.5
>mysql==5.5+
>vue2

## Build Setup

Setup Template
$ git clone https://github.com/xinsixiangyi/django-mysql-vue.git
$ cd frontend
Setup

$ yarn install
$ pipenv install --dev && pipenv shell

#Change django and vue server address by yourself

$ python manage.py migrate
